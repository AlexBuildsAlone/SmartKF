SmartKF 定价说明

SmartKF 是一个开源项目，完全免费使用。

部署方式：
1. 自行部署 — 将代码部署到自己的服务器或云容器，只需承担 AI API 调用费用
2. RunJobs 部署 — 通过 RunJobs App Container 一键部署，享受平台提供的 AI API 额度

AI API 费用说明：
SmartKF 通过 OpenAI 兼容接口调用大语言模型。费用取决于您选择的模型和使用量。
- 如果在 RunJobs 上部署，平台会注入 API Key，包含在平台费用中
- 如果自行部署，需要自备 OpenAI API Key 或兼容的第三方 API

资源需求：
- 内存：建议至少 1GB（知识库越大需要越多内存）
- 磁盘：知识库文档 + 聊天记录，通常 1GB 以内足够
- CPU：单核即可运行，2 核更流畅
