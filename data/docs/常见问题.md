# 常见问题

## 基础问题

### SmartKF 支持哪些文档格式？
目前支持 TXT 纯文本和 Markdown（.md）两种格式。将文档放入 /data/docs/ 目录，重启应用即可加载。

### 如何更新知识库？
修改 /data/docs/ 目录下的文件后，重启应用即可重新加载知识库。暂不支持热更新。

### AI 回答不准确怎么办？
请检查以下几点：
1. 知识库文档是否包含相关内容
2. 文档描述是否清晰、准确
3. 尝试优化文档结构，使用清晰的标题和段落

### 如何查看聊天记录？
访问 /admin 页面，输入管理密码即可查看所有对话记录。

## 部署相关

### 如何在 RunJobs 上部署？
1. 将代码推送到 GitHub 仓库
2. 在 RunJobs 创建 App Container
3. 设置 Init Script，从 GitHub 拉取代码并启动
4. 平台会自动注入 AI API 凭证

### 如何自行部署？
1. 克隆代码仓库
2. 安装依赖：pip install -r requirements.txt
3. 设置环境变量：OPENAI_API_KEY 和 OPENAI_BASE_URL
4. 运行：python app.py

### 支持哪些 AI 模型？
支持所有 OpenAI 兼容接口的模型，包括：
- OpenAI GPT 系列
- Claude（通过兼容接口）
- 国内各厂商的兼容接口

## 安全相关

### 客服会泄露系统信息吗？
SmartKF 内置了多层安全防护，包括提示词注入防护、输入过滤、输出过滤等。AI 不会泄露系统提示词、配置信息或技术实现细节。

### 管理后台安全吗？
管理后台和 API 接口均需要密码验证。请在 config.json 中设置强密码，并定期更换。
